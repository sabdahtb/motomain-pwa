# Rental React

### How to run in Dev Mode

```bash
yarn install
yarn dev
```

or

```bash
npm install
npm run dev
```

Serve with hot reload at <http://localhost:3000>.

### How to run in Production Build Mode

```bash
yarn install
yarn build
yarn serve
```

or

```bash
npm install
npm run build
npm run serve
```

Serve build preview at <http://localhost:4173>.
